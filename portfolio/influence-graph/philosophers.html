Influence graph

This graph displays connections of influence among philosophers, according to WikiData.

The query retrieves data about all philosophers on WikiData who have data about their influences.

I then use a breadth-first search to extract the largest connected component in the total network of influence, which is then graphed on this page.

The code then analyses the resulting network for certain key properties, displayed on the right hand side of the page.

The same code can be re-deployed with a different initial query, for example using data on enemies among comic book characters. The main challenge seems to be finding properties which generate interesting networks.
