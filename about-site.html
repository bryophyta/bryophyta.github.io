About this site

This website is hosted using Github pages. I build the static site on my own computer using Jekyll, and then upload the files to Github. (Github Pages does have built-in support for Jekyll, though!) It's probably overkill for a site this simple, but it was a fun exercise to think about how the structure of the original plain HTML site might be effectively recreated using a system like Jekyll.

I have tried to make this website accessible, although this is an ongoing process as I learn more about what is required for maximal accessibility. I hope that you have found it accessible, but I am always willing to hear feedback on how this might be improved!

Thanks are due to my friend Joe for help and tips on a number of issues with the site. But all errors are my own!
